# ecnuthesis

[![](https://img.shields.io/badge/Overleaf-ECNUthesis-brightgreen.svg)](https://www.overleaf.com/latex/templates/ecnuthesis-latex-thesis-template-for-east-china-normal-university/szppdtkvgvpk)

ecnuthesis（East China Normal University LaTeX Thesis Template）是华东师范大学学位论文模板的 LaTeX 实现，旨在帮助华东师范大学的学生使用 LaTeX 成博士、硕士学位论文与本科生毕业论文的排版工作。

## 使用 ecnuthesis

### 使用环境

请安装最新的 TeX Live，并使用 XeLaTeX 进行编译。

### 获取模板

- GitHub 仓库的 [Release](https://github.com/Koyamin/ecnuthesis/releases) 中提供了开箱即用的模板，包括具体使用说明以及示例文档。具体而言，每一个 Release 包括以下内容：
    - `ecnuthesis-sample.zip`：开箱即用的模板压缩包，解压后可直接使用
    - `ecnuthesis.pdf`：模板使用说明
    - `ecnuthesis.tds.zip`：安装用压缩包
- 本模板提供了 [Overleaf 版本](https://www.overleaf.com/latex/templates/ecnuthesis-latex-thesis-template-for-east-china-normal-university/szppdtkvgvpk)，打开链接并登录后即可直接编辑。

### 使用模板

请参阅 [Release](https://github.com/Koyamin/ecnuthesis/releases) 中的 `ecnuthesis.pdf` 文件。

### 帮助

若您在使用本模版的过程中有疑问，请移步 [Discussions](https://github.com/Koyamin/ecnuthesis/discussions) 进行讨论。
