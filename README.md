ecnuthesis（East China Normal University LaTeX Thesis Template）是华东师范大学学位论文模板的 LaTeX 实现，旨在帮助华东师范大学的学生使用 LaTeX 成**博士、硕士学位论文**与**本科生毕业论文**的排版工作。

## 使用 ecnuthesis

### 使用环境

请安装最新的 TeX Live，并使用 XeLaTeX 进行编译。

### 获取模板

- GitHub 仓库的 [Release](https://github.com/Koyamin/ecnuthesis/releases) 中提供了开箱即用的模板，包括具体使用说明以及示例文档。具体而言，每一个 Release 包括以下内容：
    - `ecnuthesis-sample.zip`：开箱即用的模板压缩包，解压后可直接使用
    - `ecnuthesis.pdf`：模板使用说明
    - `ecnuthesis.tds.zip`：安装用压缩包
- 本模板提供了 [Overleaf 版本](https://www.overleaf.com/latex/templates/ecnuthesis-latex-thesis-template-for-east-china-normal-university/szppdtkvgvpk)，打开链接并登录后即可直接编辑。

### 使用模板

请参阅 [Release](https://github.com/Koyamin/ecnuthesis/releases) 中的 `ecnuthesis.pdf` 文件。

### 帮助

若您在使用本模版的过程中有疑问，请移步 [Discussions](https://github.com/Koyamin/ecnuthesis/discussions) 进行讨论。

# ecnuthesis

ecnuthesis (East China Normal University LaTeX Thesis Template) is a LaTeX thesis template for East China Normal University (ECNU). It aims to help ECNU students use LaTeX to typeset **bachelor's theses**, **master's theses**, and **doctoral dissertations**.

## Using ecnuthesis

### Environment

Please install the latest TeX Live distribution and compile with XeLaTeX.

### Obtaining ecnuthesis

- The [Release](https://github.com/Koyamin/ecnuthesis/releases) page on GitHub provides ready-to-use templates, including detailed usage instructions and sample documents. Specifically, each Release includes the following:
    - `ecnuthesis-sample.zip`: A ready-to-use template archive that can be used directly after extraction
    - `ecnuthesis.pdf`: Template usage documentation
    - `ecnuthesis.tds.zip`: Installation archive
- This template also has an [Overleaf version](https://www.overleaf.com/latex/templates/ecnuthesis-latex-thesis-template-for-east-china-normal-university/szppdtkvgvpk). Open the link and log in to start editing directly.

### Using ecnuthesis

Please refer to the `ecnuthesis.pdf` file in the [Release](https://github.com/Koyamin/ecnuthesis/releases).

### Help

If you have any questions while using this template, please visit the [Discussions](https://github.com/Koyamin/ecnuthesis/discussions) page.
